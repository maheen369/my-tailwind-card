## 📂 Project Structure & Pages

1. **Main Directory (`index.html`)** - The central entry point of the project that acts as a modern landing card/hub, allowing users to seamlessly navigate to different projects.

2. **Weather API App (`Weather.html`)** - A functional web application designed to fetch and display real-time weather details.

3. **Professional Portfolio (`Profile.html`)** - My personal developer portfolio displaying a clean bio, key frontend skills, and quick-action social connection buttons (Instagram, WhatsApp, and LinkedIn).

---

## 🛠️ Technologies Used
- **HTML5:** For semantic web structure.
- **CSS3:** Custom styles, flexbox layout, gradients, and responsive mobile-first designs.
- **Font Awesome:** Modern icons for social buttons and tech badges.
- **Acode:** Mobile code editor used for local development and testing.
